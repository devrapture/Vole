# Documentation

Welcome to the vole-clean documentation. vole-clean is a CLI tool that scans your frontend project, finds image files that are never referenced in code, and safely removes them.

## Getting Started

- [**Installation**](/docs/installation) — Install vole-clean via npm, npx, Go, curl, or build from source.
- [**Usage**](/docs/usage) — Learn the basic commands and CLI flags.

## Reference
